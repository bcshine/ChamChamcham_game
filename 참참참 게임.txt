제목 아래 또는 주변에 귀여운 고양이 캐릭터가 웃는 기본 표정으로 등장합니다. 



kor.pngtree.com
배경은 밝고 단순하며, 고양이 캐릭터와 어울리는 파스텔톤 색감입니다.

화면 하단에 "단 1회 게임입니다."라는 안내 문구와 함께 게임 시작 버튼이 있습니다.

UI 텍스트:

게임 제목: "참참참 게임"

안내 문구: "단 1회 게임입니다."

버튼: [ 게임 시작 ]

플레이어 액션:

[ 게임 시작 ] 버튼을 탭합니다.

시스템 액션:

'게임 플레이' 화면으로 전환합니다.

사운드:

SFX: [ 게임 시작 ] 버튼 탭 시, 부드러운 "뿅!" 또는 "클릭" 효과음.

2. 게임 플레이 화면 (Gameplay Screen)
2.1. 게임 시작 / 대기 상태 (Game Start / Idle State)
화면 설명:

화면 중앙에 귀여운 고양이 캐릭터가 기본 표정으로 플레이어를 바라보고 있습니다. 고양이 얼굴은 화면 높이의 약 30%를 차지하며 매우 귀엽게 표현됩니다.

화면 상단에 "준비!" 또는 유사한 상태 텍스트가 표시될 수 있습니다.

화면 하단 중앙에 [ 참참참! 시작 ] 버튼이 활성화되어 있습니다.

그 아래에는 [ 왼쪽 ]과 [ 오른쪽 ] 방향 선택 버튼이 항상 표시되며, 이 단계에서는 아직 입력을 받지는 않지만 시각적으로는 활성화된 것처럼 보일 수 있습니다 (또는 명확한 구분을 위해 [ 참참참! 시작 ] 버튼 누르기 전까지는 비활성화 스타일 유지).

UI 텍스트:

상태 텍스트 (선택 사항): "준비!"

활성화된 버튼: [ 참참참! 시작 ]

표시되는 버튼 (초기 상태): [ 왼쪽 ], [ 오른쪽 ] ([ 참참참! 시작 ] 누르기 전까지 비활성화 또는 활성화 상태로 표시 - 개발 시 결정)

플레이어 액션:

[ 참참참! 시작 ] 버튼을 탭합니다.

시스템 액션:

플레이어가 [ 참참참! 시작 ] 버튼을 탭하면, 즉시 '2.2. "참참참!" 순간' 상태로 전환하며 "참참참" 시퀀스를 시작합니다.

[ 참참참! 시작 ] 버튼은 비활성화되거나 숨겨질 수 있습니다.

이때부터 [ 왼쪽 ]과 [ 오른쪽 ] 방향 선택 버튼이 완전히 활성화되어 플레이어의 입력을 받습니다.

사운드:

SFX: [ 참참참! 시작 ] 버튼을 탭할 때, 가벼운 "준비!" 느낌의 효과음.

2.2. "참참참!" 순간 (The "Cham Cham Cham!" Moment - 결정 및 입력)
화면 설명 (시작 시):

[ 참참참! 시작 ] 버튼을 눌러 "참참참" 시퀀스가 시작된 상태입니다.

고양이 캐릭터는 정면을 보고 있으며, "참참참" 시퀀스가 진행됩니다.

[ 왼쪽 ]과 [ 오른쪽 ] 버튼은 화면에 활성화된 상태로 표시되어, 플레이어가 언제든지 누를 수 있습니다.

UI 텍스트 (초기):

(기존 UI 요소들 유지)

시스템 액션 및 사운드 (시퀀스 진행):

첫 번째 "참!" (화면에 크게 노출됩니다)

시스템 액션:

화면 정중앙에 "참" 글자가 크고 명확하게 약 0.5초간 표시되었다 사라집니다.

고양이가 살짝 움찔하거나 눈을 한번 깜빡이는 등 가벼운 준비 모션을 보여줄 수 있습니다.

사운드 (SFX): "참!" (첫 번째 박자, 명확하고 경쾌하게)

두 번째 "참!" (첫 번째 "참" 텍스트 사라진 후 약 0.5초 뒤, 화면에 크게 노출됩니다)

시스템 액션:

화면 정중앙에 "참" 글자가 다시 약 0.5초간 표시되었다 사라집니다.

고양이가 다른 가벼운 준비 모션 (예: 귀를 쫑긋)을 보여줄 수 있습니다.

사운드 (SFX): "참!" (두 번째 박자, 첫 번째와 비슷하거나 약간 다른 톤으로)

세 번째 "참!" (두 번째 "참" 텍스트 사라진 후 약 0.5초 뒤)

시스템 액션 (이 순간 동시에 발생):

화면 정중앙에 "참!" 글자가 강조되어 (예: 느낌표와 함께) 약 0.5초간 표시되었다 사라집니다.

고양이 AI는 내부적으로 자신의 고개 방향(왼쪽 또는 오른쪽 중 하나)을 즉시 랜덤하게 결정합니다. (이 결정은 아직 화면에 애니메이션으로 표시되지 않습니다.)

플레이어는 이 세 번째 "참!" 타이밍에 맞춰 (또는 그 이전에 이미) [ 왼쪽 ] 또는 [ 오른쪽 ] 버튼을 누른 상태일 수 있습니다. 시스템은 세 번째 "참!"이 끝나는 시점의 플레이어 최종 선택을 기준으로 판정합니다.

매우 짧은 시간(예: 0.5초~0.7초, 세 번째 "참!" 텍스트가 사라진 후) 동안 플레이어의 최종 입력이 확정됩니다. 이 시간 이후의 입력 변경은 반영되지 않을 수 있습니다. (이 시간은 타이머 바 등으로 시각적으로 표시될 수 있습니다.)

플레이어 액션 (이 순간):

세 번째 "참!" 텍스트가 표시되는 타이밍에 맞춰, 이미 활성화된 [ 왼쪽 ] 또는 [ 오른쪽 ] 버튼 중 하나를 (아직 누르지 않았다면) 빠르게 탭하거나, 이미 누르고 있던 선택을 유지합니다.

사운드 (SFX): "참!" (세 번째 박자, 가장 강조되거나 결정적인 느낌의 톤으로. 예를 들어 "참!" 뒤에 "!"가 붙는 느낌의 임팩트)

사운드 (SFX, 타이머): (선택 사항) 이때부터 타이머 진행 효과음 (예: "똑-딱" 또는 긴장감 있는 소리)이 짧게 재생됩니다.

시스템 액션 (세 번째 "참!" 직후 / 입력 시간 종료 후):

세 번째 "참!" 시퀀스 종료 시점의 플레이어의 [ 왼쪽 ] 또는 [ 오른쪽 ] 버튼 선택 방향을 최종적으로 감지합니다.

제한 시간(세 번째 "참!" 이후의 짧은 확정 시간) 내에 플레이어가 버튼을 선택하지 않았거나, 이전에 선택하지 않은 상태로 시간이 지나면 '시간 초과' 또는 '선택 없음'으로 판단합니다.

'2.3. 결과 공개' 상태로 즉시 전환합니다.

(개발 참고: 중요한 것은 '참','참','참', 이 글자가 화면에 순차적으로 하나씩 크게 나타나야 플레이어가 세 번째 '참!' 타이밍에 맞춰 방향 버튼을 선택할 수 있습니다. 방향 버튼은 '참참참! 시작' 버튼 누른 직후부터 활성화되어 있어야 합니다.)

2.3. 결과 공개 (Result Reveal)
화면 설명:

(플레이어 선택 시) 플레이어가 선택한 버튼이 시각적으로 피드백됩니다 (예: 눌린 효과).

고양이가 '세 번째 참!' 순간에 내부적으로 결정했던 방향 (왼쪽 또는 오른쪽)으로 고개를 "휙!" 돌리는 애니메이션을 즉시 보여줍니다. (예: 왼쪽으로 고개를 돌린 고양이 얼굴)

고양이의 고개 방향과 플레이어가 선택한 방향을 비교하여 결과(승리/패배/시간 초과)가 화면 중앙에 텍스트와 함께 명확하게 표시됩니다.

고양이 캐릭터는 결과에 따라 다른 표정 애니메이션을 보여줍니다.

플레이어 승리 (방향 일치): "승리!", "성공!", "통했다냥!" 등의 텍스트. 고양이는 눈을 동그랗게 뜨고 놀라거나, 아쉬워하는 귀여운 표정. 



blog.naver.com
플레이어 패배 (방향 불일치): "패배!", "아깝다냥!", "내가 이겼다냥!" 등의 텍스트. 고양이는 의기양양하게 웃거나, 장난스럽게 혀를 내미는 표정. 



m.notepet.co.kr
시간 초과 패배: "시간 초과!", "너무 늦었다냥!" 등의 텍스트. 고양이는 고개를 갸우뚱하거나 어리둥절한 표정. 



www.istockphoto.com
UI 텍스트 (결과에 따라):

승리 시: "승리!" / "일치!"

패배 시: "패배!" / "불일치!"

시간 초과 시: "시간 초과!"

플레이어 액션:

결과를 확인합니다.

시스템 액션:

승리 시 최종 결과를 "성공"으로 기록합니다. (패배/시간 초과는 "실패" 또는 해당 상태로 기록)

잠시 (예: 1~1.5초) 결과를 보여준 후, '3. 게임 오버 화면'으로 즉시 전환합니다.

사운드 (결과에 따라):

SFX (고양이 모션): 고개가 돌아가는 "휙!" 또는 "슈욱-" 효과음.

SFX (승리): 밝고 경쾌한 "딩동댕!" 또는 "반짝!" 효과음. 고양이의 기분 좋은 "야옹~♪" 소리.

SFX (패배): 힘 빠지는 "삐익-" 또는 "털썩..." 효과음. 고양이의 의기양양한 "냥!" 또는 아쉬워하는 "야옹..." 소리.

SFX (시간 초과): "땡!" 또는 경고음 느낌의 효과음.

3. 게임 오버 화면 (Game Over Screen)
화면 설명:

"게임 종료!" 또는 "수고했다냥!" 같은 메시지가 화면에 표시됩니다.

플레이어의 최종 결과가 표시됩니다. (예: "결과: 성공!", "결과: 아쉽지만 실패!", "결과: 시간 초과")

고양이 캐릭터가 플레이어를 격려하거나 아쉬워하는 특별한 애니메이션을 보여줍니다. (예: 박수를 쳐주거나, 함께 슬퍼하는 모습) 



ko.ac-illust.com
화면 하단에 게임을 다시 시작하거나 타이틀로 돌아가는 버튼이 표시됩니다.

UI 텍스트:

메인 메시지: "게임 종료!"

결과 표시: "결과: [성공! / 실패! / 시간 초과!]" (상황에 맞는 텍스트)

버튼: [ 다시 도전! ], [ 타이틀로 ]

플레이어 액션:

[ 다시 도전! ] 또는 [ 타이틀로 ] 버튼 중 하나를 탭합니다.

시스템 액션:

[ 다시 도전! ] 선택 시: 게임 상태를 완전히 초기화하고 '2.1. 게임 시작 / 대기 상태'로 돌아가 새로운 단일 게임을 시작합니다.

[ 타이틀로 ] 선택 시: '1. 타이틀 화면'으로 돌아갑니다.

사운드:

BGM: 게임 오버를 알리는 짧고 독특한 멜로디가 재생되거나, 타이틀 화면의 BGM으로 전환됩니다.

SFX: 각 버튼 탭 시 효과음.

SFX (선택 사항, 성공 시): "와!" 하는 함성 또는 축하 폭죽 소리.

주의사항
세번째 '참' 순간에,  
플레이어는 오른쪽, 왼쪽 버튼을 누른다. 
이 순간에, 고양이 얼굴도 '오른쪽', 또는 '왼쪽' 으로 얼굴이  돌아가는 것을 보여다오. 
그래야 이겼는지, 졌는지 감각적으로 알수 있기 때문이다. 

고양이  얼굴방향과 버튼 방향이 같으면 플레이어 승리
고양이  얼굴방향과 버튼 방향이 다르면 플레이어 패배이다. 

게임이 끝나고, 
플레이어 승리
플레이어 패비를 한가운데,  팝업시켜서, 크고 명확하게 보여준다.
